This folder gives the different files use to test the performance of the 
Ultraleap hand tracking software : Gemini. 
The goal was to check if the performance of Gemini was good enoug to switch from 
Orion to Gemini. 
The RecordPlayBack sample will record the bone joints coordinates of each finger and will save them in a .txt file 
Thos can be later analyze on the matlab file : Gemini_analyze. 

In the folder Gemini and Orion, we record the same movements using Orion and Gemini software and we plot the results.
